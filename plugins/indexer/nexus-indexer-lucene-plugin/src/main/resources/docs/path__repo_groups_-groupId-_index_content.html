/repo_groups/{groupId}/index_content

Mount Point: /service/local/repo_groups/{groupId}/index_content

GET

Get the index content from the specified group at the specified path. Note that appended to the end of the url should be the path that you want to retrieve index content for. i.e. /content/org/blah will retrieve the content of the index at that node.

groupId The group id to retrieve index content from. path
